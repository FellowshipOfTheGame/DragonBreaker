Ao importar um sprite, ele precisa de alguns ajustes para funcionar com o Pixel Perfect:

Selecione a imagem e mude os seguintes parametros no Import Settings:
- Pixels Per Unit: 32
- Filter Mode: Point (no filter)
- Compression: None

----------------------------------------------------------------------
Para botar os sprites de plataforma no tilemap, tem um Tile Palette "Plataforms". Se nenhuma janela abrir, vá em "Window->Tile Palette" para editar.

- Nessa janela, clicar no botão "Edit" e apagar os placeholder que estão lá

- Selecionar todos os sprites separados e arrastar para o Tile Palette.

- Desativa a função edit. Agora é só selecionar a ferramenta do pincel (Brush), selecionar o tile e sair pintando na scene
